# Face Attendance System

A Face Recognition Based Attendance Management System built using Python, Flask and OpenCV.

## Features

- Face Recognition Attendance
- Student Registration
- Attendance Reports
- Admin Dashboard
- Web Interface
- Attendance Database

## Technologies Used

- Python
- Flask
- OpenCV
- SQLite
- HTML
- CSS

## Run Project

```bash
pip install -r requirements.txt
python app.py
```
